
  ETHANOL CONTENT ANALYZER - ESP32-C3 SUPER MINI WIRING DIAGRAM
  ==============================================================

  FLEX FUEL SENSOR (Continental / GM 13577429)
  +------------------+
  | Pin 1: 12V Power |-----> Vehicle 12V (ignition switched)
  | Pin 2: Ground    |-----> Vehicle Ground
  | Pin 3: Signal    |--+--> 5V Square Wave Output (50-150 Hz)
  +------------------+  |
                        |
                        |
  INPUT CIRCUIT         |    VOLTAGE DIVIDER (5V -> 3.3V)
  (Sensor -> ESP32)     |
                        |
                        +---[R1 10kΩ]---+----> ESP32-C3 GPIO4
                                        |
                                       [R2 20kΩ]
                                        |
                                       GND


  ESP32-C3 SUPER MINI
  +-----------------------------+
  |          USB-C              |
  |       +--------+            |
  |  5V  -|        |- GND       |
  |  GND -|        |- 3V3      |
  |  3V3 -|        |- GPIO10   |
  |  GP4 -|  C3    |- GPIO9    |
  |  GP5 -|        |- GPIO8    |
  |  GP6 -|        |- GPIO7    |
  |  GP7 -|        |- GPIO6    |
  |  GP0 -|        |- GPIO5    |
  |  GP1 -|        |- GPIO4  <---+-- Sensor signal (via divider)
  |       +--------+            |
  +-----------------------------+

  Note: GPIO4 is on the right side of the board.
  Power the board via USB-C (for serial monitor) or 5V pin.


  POWER SUPPLY
  ============
  Option A: USB-C from a USB power bank or car USB adapter
            (easiest for testing / prototyping)

  Option B: 5V voltage regulator (LM7805 or DC-DC buck) from vehicle 12V
            connected to the 5V pin on the board

  The Flex Fuel Sensor is powered directly from vehicle 12V.
  All grounds must be connected together (sensor, ESP32, vehicle).


  COMPLETE PARTS LIST
  ===================
  1x  ESP32-C3 Super Mini
  1x  Continental Flex Fuel Sensor (GM 13577429 or similar)
  1x  R1 - 10kΩ resistor (voltage divider)
  1x  R2 - 20kΩ resistor (voltage divider)
  1x  USB-C cable (for serial monitor + power)
  Optional: 5V voltage regulator if not using USB power


  PIN SUMMARY
  ===========
  ESP32-C3 GPIO4  <-- Sensor signal (through 10k/20k voltage divider)
  ESP32-C3 GND    --- Common ground with sensor and vehicle
  ESP32-C3 5V     <-- Power input (USB or 5V regulator)


  SERIAL OUTPUT (115200 baud)
  ===========================
  Prints to USB serial:
    Period (us): 12500  Frequency: 80.0 Hz  Ethanol: 30.0%  Duty Cycle: 50.0%  Fuel Temp: 22.5 C
